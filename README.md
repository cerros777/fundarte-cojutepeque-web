# FundArte Cojutepeque - Landing Page

Una página web moderna, interactiva y escalable para FundArte Cojutepeque, una organización sin fines de lucro dedicada a promover el arte y la cultura en la comunidad de Cojutepeque, El Salvador.

## 🎨 Características

- **Diseño Moderno**: Interfaz limpia y profesional con animaciones suaves
- **Totalmente Responsivo**: Optimizado para dispositivos móviles, tablets y desktop
- **Interactivo**: Navegación fluida, formularios funcionales y efectos visuales
- **Escalable**: Estructura preparada para agregar más páginas y funcionalidades
- **Accesible**: Cumple con estándares de accesibilidad web
- **Rápido**: Optimizado para rendimiento y velocidad de carga

## 📁 Estructura del Proyecto

## 🎨 Personalización

### Colores
Los colores principales están definidos en variables CSS en `assets/css/style.css`:

```css
:root {
    --primary-color: #2c5aa0;    /* Azul principal */
    --secondary-color: #f39c12;  /* Naranja */
    --accent-color: #e74c3c;     /* Rojo */
    --text-dark: #2c3e50;        /* Texto oscuro */
    --text-light: #7f8c8d;       /* Texto claro */
}
```

### Tipografías
- **Títulos**: Playfair Display (elegante y formal)
- **Texto**: Inter (moderna y legible)

### Contenido
Para cambiar el contenido:
1. Edita `index.html` para texto e imágenes
2. Modifica `assets/css/style.css` para estilos
3. Ajusta `assets/js/main.js` para funcionalidades
